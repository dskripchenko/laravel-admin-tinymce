# dskripchenko/laravel-admin-tinymce

> 🌐 [English](README.md) · [Русский](README.ru.md) · **Deutsch** · [中文](README.zh.md)

Alternative WYSIWYG-Engine auf Basis von TinyMCE. Eigener Lizenzschlüssel erforderlich (TinyMCE ist GPL/kommerziell).

Ein Sister-Pack für [`dskripchenko/laravel-admin`](https://github.com/dskripchenko/laravel-admin).

## Installation

```bash
composer require dskripchenko/laravel-admin-tinymce
php artisan migrate
```

Das Plugin registriert sich automatisch über Laravel Package Discovery.
